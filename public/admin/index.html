🚀 Como funciona a Instalação?

1. Instalação Automática (Recomendado)

Ao aprovar a instalação deste app na Olist/Vnda, nosso servidor automaticamente registra os scripts necessários via API na sua loja. Você não precisa mexer em códigos!

2. Criar a Página da Lista de Desejos

No painel da Vnda, acesse Conteúdo > Páginas > Nova Página. Defina o título como "Minha Lista de Desejos" e a URL (slug) obrigatoriamente como wishlist. Deixe o conteúdo em branco e salve. O app renderizará toda a lista ali dinamicamente.

3. Ícone com Contador no Header

Para adicionar o ícone de coração com contador no seu menu, basta inserir um elemento com a classe CSS wishlist-count onde preferir no seu cabeçalho (por exemplo, ao lado do carrinho de compras). O script atualizará a quantidade favorita dinamicamente.
